收集全世界的視覺語言 － The Noun Project

Posted 14 5 月, 2015 by infographics.tw under 介紹

一圖勝萬語，有時萬言書不如一張圖來得巧。符號、圖標已成生活不可或缺的一部份，但設計圖示又不是簡單的工作；若有個地方已經收集了世界上所有這樣的視覺語言該多好？沒問題，讓 The Noun Project 為你服務。

或許已經有不少朋友聽過 The Noun Project 了，簡單的說，它是個向量圖示 / 符號的分享網站，我們可以透過文字搜尋找到數以千計的對應圖示，所有的圖示都同時提供單色的向量 SVG 檔與點陣式 PNG 格式，以及對應的使用授權。

The Noun Project 如何維護這麼大量的圖示資源呢？他提供了上傳圖示的管道，讓全世界的設計師都可以上傳自己的圖示作品。那麼問題來了，是誰要這麼熱心來上傳這些圖呢？這就要談到圖示使用與授權模式了。

網站中的圖示大多以 Public Domain 或 CC-BY 3.0 的授權方式提供，這兩個授權分別代表：

  • Public Domain: 任意使用，不需標註作者，沒有任何限制
  • CC-BY 3.0: 只要在使用時標註原作者就可以任意使用 ( 包含改作、商業利用等等 )

簡單的說，基本上這個網站裡的圖示我們都可以免費使用，只要在使用 CC-BY 3.0 授權的作品時有適當的註明原作作者即可。

這個圖示「Pie Chart」的授權為「CC BY 3.0」

 

當姓名標註是個問題時， The Noun Project 提供了進階版的付費帳號，一旦我們將帳號升級，所取得的圖示就不再需要標註作者，而作者也能從我們的月費中獲得合理的分潤。聽起來是個多贏的局面，一般用戶有完整的圖示庫可以使用、圖示設計師也可以獲得合理的酬勞。

進階帳號的月費目前是 9.99 美金，如果我們只是偶而需要使用，也可以考慮花 1.99 美金取得單張圖示授權的方式。當然，前提都是我們不想標註作者姓名的情況下囉！若要進行姓名標註，可以參考 The Noun Project 提供的「姓名標註須知」，基本上除了圖示名稱、作者外，我們還需要註明圖示來源是 The Noun Project ；該須知內還針對不同的使用方式 ( 影片、海報、商業利用等等 ) 分別做不同的說明，但準則大致都相同，在使用前不妨仔細看一下囉。

若不是付費帳號，在下載之前仍可以選擇購買單張授權

 

使用圖示

在我們有圖示需求時，直接上 The Noun Project 網站註冊或登入，然後利用搜尋功能瀏覽適合的圖示。確認圖示的授權後，我們可以選擇是要 SVG 向量格式或是 PNG 點陣圖檔，端看我們利用的需求。

除了自己下載檔案外，最近 The Noun Project 推出了一款 Mac 專用的桌面應用程式，讓我們可以直接在電腦中搜尋圖示，然後用拖拉的方式丟到繪圖軟體中使用，相當方便。而且不光是繪圖軟體，只要接受拖曳檔案開啟的軟體或網站都可以這樣使用，所以像是 Piktochart 線上資訊圖表工具因為接受拖曳檔案上傳，所以我們也可以結合 The Noun Project 一起使用，瞬間圖示庫就擴充了 N 倍！

不過，用拖曳取得的圖示授權要怎麼處理呢？Noun Project for Mac 其實算是進階帳號的一部份，但在我們第一次下載安裝時， The Noun Project 也同時贈送 14 天進階帳號試用，在這 14 天內使用的圖示都會自動的取得使用授權，不用另外進行姓名標示囉！不過 14 天之後也就跟著不能使用了。雖然如此還是很方便的一個工具，想嘗鮮的朋友不妨試一下。
