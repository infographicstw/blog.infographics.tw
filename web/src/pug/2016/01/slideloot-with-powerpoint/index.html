不懂設計？用 Slideloot + Powerpoint 快速製作資訊圖表

Posted 20 1 月, 2016 by infographics.tw under 介紹

不擅長繪圖軟體也不會畫畫，卻又想要做出不錯的資訊圖表嗎？現在資訊圖表工具百百種，但其實光是 PowerPoint 就能做出很不錯的資訊圖表，而簡報樣版網站 Slideloot 提供了近千種的免費資訊圖表樣版，兩者的搭配讓我們不再需要擔心軟體與技術門檻，只要專注於做好內容！

( 2016/01/31 編按：該網站已改版，現為 Box-a-Vox (boxavox.com) ，介面混亂、速度很慢、並且似乎沒有提供免費資訊圖樣版了，請讀者注意。若 Box-a-Vox 有後續改進，我們會持續關注並做更新。 )

Austin Kleon 所著「Steal Like An Artist」一書中提到，幾乎大多設計與創意最初都是由參考或學習其它人的創作開始的。現今有許多網站匯集了全球各地的各種創意創作作品，例如 Dribbble 、 Behance 、 500px  、 DeviantArt 等等網站，其實很多時候我們缺乏靈感的時候，去上頭晃晃都能有意外的收獲呢！

然而設計不光只是靈感的部份，很多時候我們受到一些現實技術的限制，只能看著別人好棒棒的作品流口水，即便想得到內容，也做不出來。其實很多時候，現成的樣版也可以幫上我們不少忙，裡面包含了各式各樣即時可用的視覺元素、資訊設計編排等等，我們需要的除了將之妥善組合之外，就是將選出恰當的樣板與我們的資料再做加工了。

在加工方面，雖然專業繪圖軟體功能相當強大，但往往我們並不需要這麼多的功能；以 Adobe Illustrator 來說，他讓我們可以做任意線條的描繪、形狀的組合、透視、網格、像素化甚至 3D 功能，超級強大，但製作資訊圖表的時候，不一定要用到這些複雜的功能，我們也可以作出很棒的作品。

Microsoft PowerPoint 做為一個簡報軟體，設計得很容易讓我們可以快速的做資訊與視覺的安排，同時其普及性、知名度與親切的價格，比起專業繪圖軟體來說，對入門或初學者都是更好的選擇，只要搭配適當的樣版，我們也可以很快速做出有質感的資訊圖！

Slideloot.com 網站提供了相當大量的 PowerPoint 簡報樣板，其中有近千個的資訊圖表樣版可以供我們免費下載使用，這對想要製作資訊圖表的朋友來說是不是一個福音呢？

首先連至 Slideloot.com 的資訊圖表樣板列表，馬上就可以看到很多樣版供選取：

這邊的樣板都是可以免費下載的，但條件是我們需要註冊並追蹤 slideloot 的社群帳號。點擊我們有興趣的樣板後可以看到右方的下載資訊，寫道「The Slide is Locked」的字樣：

就提供了大量免費樣板的立場來說，要求按個讚不過份吧。不過按讚之後仍然需要註冊並登入才能下載樣板。下圖為我們成功下載後利用 PowerPoint 打開樣板之後的截圖：

可以看到其中的元素都是可以編輯的，有需要的話各別色塊的顏色、大小與形狀，文字的字形與位置都可以調整。

由於下載的樣板很像是攝影中常見的「光圈」符號，就讓我們來試試看做一個簡單的攝影資訊圖。我們試著利用 loading.io 色票生成器產生「風景攝影」主題的色票：

然後將之套用到樣板中的色塊，同時將文字修改成適合這個主題的內容，改成「八個不可不知的攝影要素」，做出的成品如下：

其中圖示並沒有換過，所以與主題有些脫勾。這裡我們可以使用先前介紹過的「Noun Project」(請參考 收集全世界的視覺語言 － The Noun Project 一文 ) 搜尋我們需要的圖示來代換，難度不高，在這裡就不另外作示範。

結語

製作優秀的資訊圖表往往牽涉到相當多層面的問題，包含整體視覺設計、資訊安排、資料搜集與整理、呈現媒介的考量等等… 往往不是像我們這裡寫的這麼簡單的幾個步驟。然而，很多時候我們只是缺乏一些技巧或工具，若因此而做不出想要的成品，不是很可惜嗎？

Slideloot.com 提供數量可觀的免費樣板供使用，同時這些樣板的風格都是一致的，所以交互搭配使用非常容易；而 Microsoft Powerpoint 價格不高、使用容易，且在學校、公司等場所往往已是預裝軟體，對學生或是一般需要製作簡單資訊圖的上班族來說已是相當方便。所以說，你想要嘗試看看製作簡單的資訊圖嗎？技術細節不用再考慮了，現在就開始從主題跟文案開始發想吧！
