不只閱讀，更能看見－革命性的留言系統 pol.is

Posted 3 4 月, 2015 by infographics.tw under 介紹

這個標題下得可能不好，因為 pol.is ( polis.io ) 不光是個留言版，而是包含「線上評論」與「觀點分析」的「視覺化」系統。他的理念在有意義地「整合巨量討論」同時仍「保留非主流的意見」，並且能引導讀者以更積極且深入思考的方式「參與討論」。

polis.io 的線上範例

一般的留言系統例如臉書、DISQUS 等等讓使用者可以留下訊息、針對特定對象回覆或評價，然而評論組織的思維通常是線性或樹狀的。我們常常看到一些線上討論群組有 threading 的概念，或者是在標題中一大堆的「Re:」，這代表討論串像葡萄般一串一串的，你必須用走迷宮的方式逐條追蹤，即使有些討論已經叉開了話題或是根本無關緊要。

雜亂的評論讓人眼花撩亂，也無法讓人清楚理解目前意見的走向。為了解決這個問題， pol.is 以更結構化的方式整理評論，並利用視覺化的手法清楚的提供我們每個人的意見傾向。你已經在上面的圖看到他所謂的視覺化了，每個人佔據圖表的一角，距離就代表了彼此想法之間的差異。

pol.is 利用 API 型式供我們在網站上置入元件，元件則分成三個主要部份：

  • 評論 ( Write ) －以文字提出你的見解
  • 回應 ( React ) －透過支持與否表達你的立場
  • 概觀 ( Majority ) －視覺化呈現所有人之間的立場偏好，同時包含主流與非主流的意見

類似臉書一般人偏好按讚更勝於評論，pol.is 在評論發表之後，隨機的出現在「React」中，其它沒有寫作評論的讀者則以「同意」、「反對」、「略過」三擇一的動作來表達意見。因為這樣機械化的動作很容易進行讓人更傾向去做，而就像做心理測驗一樣，透過你不斷的點擊，程式最終會在「Majority」中找出你的想法在群體中的位置。

除了展示各人在群體評論中的定位外，我們同時能看到 pol.is 將所有人依意見傾向分成數個群組。類似想法的人被分到同一組中，想法比較獨特 ( 或是來亂的 ) 的人被獨立出來；當然不可能群組內的所有人想法都一樣， 所以 polis 會針對各別評論顯示群組內的支持率：

從上圖你可以看到左邊的主流群組有 86% 的人都支持「 Football is ‘too big to fail’ 」，而右邊的少數群體則有 72% 的人不贊同。除了這樣單一的意見外，所有意見則在左方表中由支持率從左往右排列，你可以點擊左右箭頭按鈕來瀏覽不同意見的支持狀態，甚至可以藉由地圖瀏覽各支持者所在的地理位置。

pol.is 這樣的設計讓你以更宏觀的角度來理解群眾的立場，而非僅以「熱門留言」「最新留言」的方式來看事情。我們以往傾向於尋找最多人按讚、最多網友推文的內容，但相信大家都有被「騙進來」的經驗，也曾經在文章底下「反詐騙」，有時甚至會看到「一樓救了原PO」。

很多時候少數意見並不代表不存在、不重要、不正確，但現今的社交平台傾向於讓你只看到你喜歡的內容，也就是，那些你認為正確、重要、真實存在的內容。那反對的聲音？不認同的意見？這些對於反思己身很重要的看法，很容易就被過濾掉而從你的動態牆中消失。

隨著網路興起與科技的發達，越來越多事情的做法可以完全不同於一百年前的人類；投票、評論、表達意見也不再需要透過投票所、郵寄等等的方式來進行。視覺化的力量也是一樣，從手繪，紙本印刷、數位圖片、互動內容到協作圖表，技術與藝術不斷的在進步，如何利用這樣的進步在各種領域上，相信資料視覺化的愛好者也應該認真思考這個問題。
