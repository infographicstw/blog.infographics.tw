他的插畫說你的故事 — 用 Storybird 創作視覺故事

Posted 14 12 月, 2015 by infographics.tw under 介紹

喜歡說故事，卻不善於繪畫與設計嗎？「要是有人能幫我畫就好了…」那你真的該看看這個服務！Storybird 串連起插畫家社群，讓無盡想像的插畫為你所用，說出你自己的故事。一起來看看 Storybird 怎麼做到這件事吧！

不可否認，圖文書是很棒的敘事方法。從小我們看的就是圖畫書，指著圖中的小鳥或星星問父母：「這是什麼？」視覺式的概念表現不光是應用在資料表現上，以漫畫、插畫等視覺手法傳達概念或理念的例子也是比比皆是，唯一的問題是：插畫創作包含了繪畫技法跟創意的含量，並沒有辦法透過程式或規則輕易的產生。想想看，Google Chart 、 Amcharts 或是 Plot.ly 會產生插畫嗎？

Storybird 解決這個問題的作法很簡單。他本身是一個說書人、閱讀人跟插畫人的串連平台，透過匯集大量的插畫人作品，已讓全世界的說書人創作超過 500 萬篇故事，再讓閱讀人可以從中挑選自己喜愛的作品，線上閱覽、印製成冊或產生畫框、週邊產品等等。

註冊並登入 Storybird 後我們從網頁上方的選單中點選「Create」，便會出現一個如上圖所示的作品列表，點擊圖片可以了解更多關於該作品的細節，比方說下圖為 inkymum 所創作的「Wise Guy」：

這時點擊閃電符號按鈕或「Use this Art」，我們便能開始製作我們的圖文書。如下圖，創作的種類有三種：

  • Longform Book ( 長篇故事 )
  • Picture Book ( 多頁短篇 )
  • Poem ( 單頁式故事 )

 

下圖即為一個短篇故事的編輯畫面。我們可以看到下方的頁面縮圖與右方的圖文編輯框，這讓我們可以透過拖拉的方式排列我們的故事書；左右散落的圖片則是同一作者的其它作品，同樣可以透過拖拉的方式放到右方的圖文編輯框中。

每頁固定只能擺放一張圖片，位置略有不同；我們可以讓插畫佔滿全頁，或者是左、右、上、下任一處留白以填入文字。上圖右方的文字即為我們自行填入的文字。

封面設計略有不同，Storybird 預設提供四種不同的封面排版形式，我們選擇其中一個以後挑選圖片、顏色並依其排版填入書名、作者等資訊。 ( 其中三種是 Premium 樣式 )

製作完成之後，我們可以存檔關閉，也可以選擇發表我們的作品。發表作品需要填入額外的詳細資訊，包括作品類別、關鍵字、適合年齡等等資訊，並且送給 Storybird 做審查：

審查時間不定，從線上作品量的規模來看，想必會等上好一陣子吧。 Storybird 的 FAQ 中即有提到審查時程，如果成為付費會員可以加快審查速度。

即便尚未審查完成，我們的作品也可以在線上閱覽了，下圖是我們製作關於視覺化使用顏色的小品，使用 pixopop 的插畫素材，簡單以小朋友能夠理解的方式介紹幾種顏色所代表的意含，若有興趣可以點擊這個書本的連結看看他長什麼樣子：

最後，無論是單張的插畫作品或是製作完成的圖文書，我們都可以分享給朋友，也可以讓其它人線上購買。Storybird 提供印刷品製作服務，包括、框畫、卡片、精裝及平裝故事本，也提供電子書或自行列印裝訂的下載格式，甚至連手機殼都接受訂購。如果你有一個好故事，不妨也把他做成一本故事書來販賣吧！

結語

這次介紹的 Storybird 稍微跳離了資料視覺化範躊，進到了圖像與敘事的領域；然而敘事在視覺化的過程中仍是相當重要的一環；看圖說故事也說明了，一樣的圖片經過不同的說書人之手，也可以有千變萬化的解釋。

資料視覺化往往給人很生硬的感覺，然而使用圖表、互動網頁程式來表現資料的同時，我們也可以利用圖文搭配來表現較概念性的想法；在沒有技術與夥伴的情況下，利用 Storybird 可以讓我們很快的做出一個成品，就像是前面我們製作的視覺化顏色應用圖文書一樣。下次若有這類的需求，別忘了考慮使用看看 Storybird 吧 🙂
