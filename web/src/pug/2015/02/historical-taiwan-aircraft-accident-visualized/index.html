歷年台灣航空事故圖表

Posted 4 2 月, 2015 by infographics.tw under 作品

2015年2月4日，一架從松山前往金門的復興航空 235 號班機疑因機件問題，機身傾斜無法順利控制，此時飛機剛駛離松山機場，尚在都會區，若是墜落後果不堪設想，機長盡力避免飛機撞擊附近大樓並試圖沿基隆河飛行，最後撞擊環東大道後墜入基隆河中。

2014，好像是個不安份的年份；常聽到友人看著電視在說，最近空難怎麼這麼多！馬航失蹤、馬公空難歷歷在目，這天又碰到了如此不幸的事情。根據維基百科，台灣自 1963 年來發生過 36 起空中事件，部份是劫機或者是非致命的事故，其它則是重大空難，我想大家一定都還記得吧，名古屋空難、大園空難以及澎湖空難，死傷人數均在兩百人以上；而自 2002 年的澎湖空難之後，其實十幾年來台灣並沒有再發生相當嚴重的空難，只是到了 2014 年，這半年的兩起意外，也已經造成了近70多人死亡、尚有近20人失蹤的慘劇了。

剛好前陣子我們在臉書分享了零傳媒製作的伊斯蘭恐怖攻擊事件圖，不如今次我們利用同一個工具 ( geoevent ) 帶大家來回顧一下台灣歷年空難事件發生的時間、位置還有規模吧！

歷年傷亡人數最多的幾次空難：1981三義空難(遠航，110人死亡)、1994名古屋空難(華航，264人死亡，地點在日本)、1998大園空難(華航，202人死亡)、2002澎湖空難(華航，225人死亡)，其中又以名古屋空難最為嚴重。可以注意到圖表中圓圈大小並沒有戲劇般的變化，因為空難往往一次的死傷人數都相當可觀，人數也多在數百人以內，並不像恐怖攻擊有數千人 ( 如 911 ) 傷亡的情況。

有些事件在圖表中的經緯度 (圓圈中心) 並不盡然正確，有些僅大概標示於事故所在鄉鎮的位置，除了橫跨數鄉鎮的例子以外，有些找不到確切座標，或是沒有準確的座標 ( 比方說空中解體 ) 。

資料來源: 維基百科 / 台灣航空事故列表
