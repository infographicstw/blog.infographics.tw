使用 infogr.am 製作圖表與資訊圖

Posted 12 2 月, 2015 by infographics.tw under 介紹

不同於一般的資訊圖表製作網站， infogr.am 是以圖表製作為主軸，以捲軸式的頁面插入各種不同的素材來做編輯，大大簡化了圖表製作的流程；也因此，infogr.am 支援的圖表數量與變化性也比其它網站還要來得多。那麼，就讓我們一起來看看 infogr.am 提供了什麼樣的服務吧！

除了可以製作圖表以外， infogr.am 正如它的名字一般，也是個很棒的資訊圖表編輯工具。 infogr.am 以圖表為中心，讓我們在做捲軸下拉的同時可以插入各式各樣的圖表、多媒體與文字。下圖展示了 infogr.am 所支援的部份圖表類型：

infogr.am 提供了莫約 14 種的圖表類型，有些類型有額外的子類別可供選擇，例如長條圖就有其它三種 堆積長條圖、分組長條圖還有圓型的長條圖，含子類別加一加總共大概也有 31 種類型可以使用，常見的類型都囊括了；不過倒是沒見到熱圖、日曆圖、圓餅泡泡圖等，但包含了許多非典型卻又經常出現在資訊圖表中的類型。

在建立專案畫面我們可以選擇不同的專案類型，以下以選擇建立圖表的情境來做介紹。建立圖表後會進入編輯畫面，編輯畫面算是相當簡明，上方的功能按鈕與右方的設計按鈕，都相當簡單易懂，中間的圖表部份以輸出紙張的設計暗示圖表可以設計得無限長度。下圖為選擇 Group Bar Chart 後大概會有的樣子：

右方的按鈕主要是讓你新增元件使用的，你可以在圖上增加一行字、插入一份地圖、插入圖表、影片或照片。直接將滑鼠移到圖表上方也可以有類似的效果：

值得一提的是，插入地圖選項裡不僅如其它一般的圖表繪製工具可以插入五大洲的資料以外，他同時也有相當多國家的細部行政區圖，比方說台灣各縣市的區塊；可惜的是這樣的功能必須付費才能使用，目前  Pro User 一個月的費用是 50 美金，不是很便宜，如果有需求的話，就自己斟酌看看囉。

雙擊圖表可以進行編輯，包括提供資料以及設定圖表的選項等等。繼承了網站整體的設計理念，設定頁面也保持著簡明的風格，在這裡你看不到太多太複雜的細節，但仍然能夠符合基本的需求，包括顯示格線、數值、設定圖表顏色與寬度、圖表顯示的範圍等等。在這邊也可以切換圖表的子類別 ( 比方說從一般的長條圖換成堆疊長條圖 ) ，不過不同類型的圖表不能切換，只能重新產生一個。

圖表的資料若存在 CSV 或 XSL 裡，可以利用匯入功能直接載入，也可以直接利用拷貝 / 貼上的功能匯入圖表；如果圖表 X / Y 軸呈現的方向不如你的預期，直接點選表格左上角的雙箭頭欄，立刻替你將資料做位置的轉換調整，相當的方便。

製作圖表的同時，infogr.am 也提供了約 8 個主題給我們套用，改變的包括基本的配色還有背景的樣式。製作完成以後，利用網頁上方的 Publish 按鈕便可以產生一個外部連結供其它人瀏覽這個作品囉！這邊有個範例利用 infogr.am 製作了一個瀏覽器市佔率的圖表：

這個圖表的網址是 https://infogr.am/2014_-48867810922 ，你也可以直接連結過去看個究竟。

優缺點總結

infogr.am 是一個可以很快速上手的工具，介面功能都可以說是相當簡明，但相對的這也帶來了一些不便：

  • 圖表部份
    • 圖表無法客製化數字的顯示方式 (  線圖中的 Y 軸格線，或是長條圖的顯示文字 )
    • 沒有提供 undo 功能，誤刪元件就完了
    • 圖例不是很好看，尤其在使用中文的時候。而且又無法客製化。
    • 圖表顯示時會帶入一個動畫，這在更新資料時也會跟著一直閃，很礙眼
  • 整體工具
    • 文字無法選擇字型或大小，只能手動置中置右
    • 刪除鍵的行為背離使用者習慣，會跳出編輯環境
    • 提供的設計素材有點太少了，付費版才有提供。

另外，完成的圖表會帶有 infogr.am 的商標，除非付費到 Pro ( = 50usd/month) 才能拿掉。事實上有很多部份都是需要付費才能有的功能，而他的基本月費 19usd 差不多等於訂閱一套 Adobe after effect 的月費了，除非真的大量需要產生這樣的圖表，不然並不能說是很便宜。而他的最大優點就是真的可以很快的做出一個成品來，而且在地圖功能上他也能夠畫出比較細部的台灣，就這點來說若臨時需要一些視覺化表現的效果，使用他做為開發的基礎，到了一個完成度時再截圖加以後製也許也是一個不錯的方式喔！
