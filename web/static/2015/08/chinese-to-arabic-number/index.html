線上阿拉伯數字與中文數字轉換工具幫你快速切換數字

Posted 30 8 月, 2015 by infographics.tw under 介紹

中文數字轉換器除了 Spreadsheet 套件，也推出了網頁版，試試看吧：

「打開一看，滿滿的中文數字！！」你是否碰過這種窘境？明明資料都到手了，卻被上千行的中文數字打敗… 沒緊張，這次為大家介紹一個好用的 Google Spreadsheet 擴充「中文數字轉換器」，幫你在國字與數字之間自由的轉換，再也不用一行一行慢慢重 Key 了！

數字可以說是視覺化最好的夥伴，但是數字也常常用捉弄人的方式出現；中隔的逗點、不同的計數單位、其中以像是「陸億九玖拾柒萬」這類的國字來表示數字也是個很令人頭大的問題。在 Microsoft Excel 中我們可以用設定數值儲存格格式或利用「NUMBERSTRING」函式來處理，不過並不是任何人都有安裝 Excel 。那麼，就讓我們來看看如何在 Google Spreadsheet 裡做到這件事吧！

Google Spreadsheet 做為一個線上試算表工具可以說是非常強大，很多 Microsoft Excel 可以做到的事他都能做。我們甚至也可以為 Google Spreadsheet 提供客製化函式，讓我們自行撰寫各式各樣的資料處理指令。

如果願意的話，這些撰寫好的指令還可以放到 Chrome Web Store 中提供其它使用者安裝使用，這次要介紹的「中文數字轉換器」即可以在 Chrome Web Store 裡面找到 ( 連結在此 ) 。你可以直接點前面這個連結來安裝，也可以開啟任意的 Google Spreadsheet 文件之後，從選單列中的 Add-ons ( 外掛程式 ) → Get Addons (取得外掛程式 ) 中開啟外掛瀏覽器；接著在對話框右方的搜尋欄中鍵入「中文數字」，應該就可以看到「中文數字轉換器」了：

這個外掛是免費的，往下拉稍微了解使用方式後，點擊右上角的「+免費」按鈕，並在接下來的權限提示畫面點選接受，你就裝好「中文數字轉換器」外掛囉！

使用中文數字轉換器

「中文數字轉換器」的使用方式也相當簡單：

  • 在試算表中，選取你想要轉換的資料範圍
  • 在選單列中點選「外掛程式」 ( 英文為 Add-on ) → 中文數字轉換器
  • 選擇你想要進行的轉換動作。目前中文數字轉換器支援以下幾種動作：
    • 中文數字轉換成阿拉伯數字 ( 中轉數 )
    • 阿拉伯數字轉換成小寫中文數字 ( 數轉中/小寫 )
    • 阿拉伯數字轉換成大寫中文數字 ( 數轉中/大寫 )
    • 小寫中文數字轉成大寫中文數字 ( 小中轉大中 )
    • 大寫中文數字轉成小寫中文數字 ( 大中轉小中 )

( 大寫中文數字範例: 壹貳參肆伍陸柒捌玖拾佰仟 ; 小寫中文數字範例: 一二三四五六七八九十百千 )

其它說明

除了基本的數字轉換，有幾點也值得特別提一下：

  • 金額結尾如「元整」「元新台幣」會被完整保留，在金額轉換上特別實用。
  • 除了金額的結尾外，若碰到含有非數字資料的欄位，則會略過不做轉換。
  • 阿拉伯數字與中文數字摻雜也是可以用的，例如 1仟兩百3十萬。
  • 「一百二」這類的省略寫法會照口語的方式翻譯成 120。

大致上就這樣了，小巧的方便工具，不曉得你是否用得上呢？碰到中文數字的時候，記得拿出來使用看看吧！
